# Graduation Project -- Simple Dashboard

Husky used for commits.

### Frontend Technologies
1. HTML
2. CSS - Preprocessor (SCSS)
3. Javascript
4. Vue JS (2.x)
5. Vue Router
6. Vuex
7. Vuelidate
8. Axios
9. i18n (for language changing)

### Backend Technologies
1. NodeJS
2. ExpressJS
3. MongoDB (with Mongoose)
4. PostgreSQL
5. Bcrypt
6. Express-Validator
7. Nodemon
8. Cors
9. JSON Web Token
